# 2Gold Node TypeScript

Du an Node.js + TypeScript dung:

- login that voi `https://2gold.biz/User/ProcessLogin`
- session auth + luu cookie/session remote
- giao dien `/Installment/Index/` theo phong cach 2Gold
- sidebar rut gon chi con:
  - Tra Gop
  - Cua Hang
  - Nhan Vien
  - Phan Quyen
  - Lich Su thao tac
- component layout dung lai cho cac trang khac

## Cai dat

```powershell
cd 2gold-node-ts
npm.cmd install
```

## Chay dev

```powershell
npm.cmd run dev
```

Mo:

- `http://localhost:3000/login`

## Route

- `/login`
- `/Installment/Index/`
- `/Shop/Index/`
- `/Staff/Index/`
- `/Staff/PermissionStaff/`
- `/History/`
